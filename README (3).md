### 🏛️ Installation — The Rite of Summoning

Choose thy path, traveler:

#### Via the Royal Courier (npm / pip / cargo)

```bash
# Node & the Holy Script
npm install -g emperorify
# or
yarn add emperorify

# Python & the Alchemist's Way
pip install emperorify

# Rust & the Forged Steel (if applicable)
cargo install emperorify
```

#### From the Imperial Archives (Source)

```bash
git clone https://github.com/tsukiyukihira01-hue/emperorify.git
cd emperorify
# Install dependencies
npm install  # or pip install -r requirements.txt
# Anoint the build
npm run build
```

> Requires **Node >= 18** or **Python >= 3.9** — As decreed by the Crown.

### 👑 Quick Start — Claim Thy Throne

#### I. The Imperial CLI

```bash
# Coronate a text
emperorify "hello world"
# => 👑 HAIL, EMPEROR OF HELLO WORLD 👑

# Bestow majesty upon a file
emperorify decree --input ./peasant.txt --style gothic --output ./imperial_edict.txt

# Full imperial ceremony
emperorify --help
```

#### II. The Arcane API

```javascript
import { emperorify } from 'emperorify';

// A simple blessing
const decree = emperorify("make me emperor");
console.log(decree);
// => "By Divine Right, Thou Art Emperor."

// With royal options
const epic = emperorify("my project", {
  style: 'baroque',
  prefix: 'Imperator',
  suffix: 'Augustus',
  gilded: true
});
```

```python
from emperorify import ascend

# Rise, peasant
scroll = ascend("my boring text", style="medieval", crown=True)
print(scroll)
```
