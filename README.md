## 🎭 Sekilas Tentang Ngawi Core
Project ini didedikasikan untuk melestarikan kebudayaan ekstrim, barbershop legendaris, dan lagu kebangsaan yang sangat *emotional*. Jika Anda terngawi-ngawi, Anda berada di tempat yang tepat.

## ⚙️ Fitur Unggulan (Ngawi Features)

* **Auto-Ambatukam Voice System:** Konversi log system error menjadi teriakan emosional penuh penghayatan.
* **Barbershop Rusdi Theme:** UI/UX super rapi dan klimis layaknya potongan rambut bintang 5 dari Mas Rusdi.
* **Anti-Normie Shield:** Memblokir request dari user yang belum paham arti dari kata "Jomok".

## 🚀 Cara Instalan (Installation)

Pastikan Anda sudah meminum air suci Ngawi sebelum menjalankan command berikut di terminal Anda:

```bash
# Clone repository suci ini
git clone [https://github.com/rongawi/ambatukam-core.git](https://github.com/rongawi/ambatukam-core.git)

# Masuk ke direktori kerajaan
cd ambatukam-core

# Install seluruh brainrot dependencies
npm install --save-dev ngawi-brainrot-toolkit

# Jalankan server utama
npm start -- --mode="ambatukam"
